Introduction

This project is designed to automatically control a lamp based on the surrounding light intensity. It uses an LDR (Light Dependent Resistor) sensor to continuously monitor the brightness of the environment. When the light level drops below a certain threshold ‚Äî such as during evening or in dark conditions ‚Äî the Arduino UNO processes the sensor data and switches the lamp ON through a relay module (or an LED for demonstration purposes). Similarly, when sufficient daylight is detected, the lamp automatically turns OFF. This system provides an efficient, hands-free solution for smart lighting control. It helps conserve energy by ensuring that lights are only used when necessary, making it ideal for use in homes, streets, and offices. Moreover, it serves as an excellent beginner-level IoT and automation project, helping learners understand how sensors, microcontrollers, and electronic components interact to create real-world applications.

Step-by-step Build Instructions

  1. Hardware Wiring
    • Place the Arduino and the mini breadboard.
    • Connect one leg of the LDR to 5V and the other leg to analog pin A0 via a resistor to ground (voltage divider).
    • Wire the relay module (or the lamp driver) to a digital pin (e.g., D8) and to an external power supply if the lamp needs higher voltage/current. For demo, you can use an LED + resistor on D8.
    • Double-check grounds: Arduino GND must be common with the sensor/relay module ground.
  2. Software Upload
    • Open the Arduino IDE, paste the sketch from the Arduino Code section, select Arduino UNO, and upload.
  3. Adjust Threshold
    • Adjust the threshold value in the sketch to match your ambient light conditions.
  4. Test
    • Cover the LDR to simulate darkness ‚Äî the lamp should switch ON. Uncover to switch OFF.

Arduino Sketch

Below is a tested example sketch that reads an LDR on A0 and toggles a lamp/relay on D8. Replace the threshold or pins to match your wiring.

// Automatic Night Lamp System
int ldrPin = A0;     // LDR connected to analog pin A0
int ledPin = 9;      // LED connected to digital pin 9

int ldrValue = 0;    // current averaged LDR value

// Adjust these thresholds for your environment
int darkThreshold = 520;   // below this = dark
int lightThreshold = 620;  // above this = bright

bool ledState = false; // stores whether LED is ON or OFF

// --- Function to get a smoothed average reading ---
int readLDR() {
  long sum = 0;
  int samples = 20;   // number of samples to average
  for (int i = 0; i < samples; i++) {
    sum += analogRead(ldrPin);
    delay(5);         // small delay for stability
  }
  return sum / samples;
}

void setup() {
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  ldrValue = readLDR();       // get smoothed reading
  Serial.println(ldrValue);   // show averaged value

  // If it's dark and LED is OFF ‚Üí turn it ON
  if (ldrValue < darkThreshold && !ledState) {
    digitalWrite(ledPin, HIGH);
    ledState = true;
  }

  // If it's bright and LED is ON ‚Üí turn it OFF
  else if (ldrValue > lightThreshold && ledState) {
    digitalWrite(ledPin, LOW);
    ledState = false;
  }

  delay(200); // short pause between checks
}

Tip: If you're using a relay module, ensure the relay coil supply and the Arduino share a common ground. For mains lamps, always follow safety rules and disconnect power while wiring.
