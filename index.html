Introduction

This project automatically controls a lamp based on ambient light. The LDR senses light level and an Arduino UNO toggles a lamp via a relay (or an LED for demo). It’s perfect for energy saving, hands-free lighting, and IoT learning.

Components

All the hardware used in the project (small icons provided):

  • Mini Breadboard
  • Jumper Wires
  • Resistor
  • Arduino UNO
  • LDR Sensor
  • Bulb (lamp / relay)

Step-by-step Build Instructions

  1. Hardware Wiring
    • Place the Arduino and the mini breadboard.
    • Connect one leg of the LDR to 5V and the other leg to analog pin A0 via a resistor to ground (voltage divider).
    • Wire the relay module (or the lamp driver) to a digital pin (e.g., D8) and to an external power supply if the lamp needs higher voltage/current. For demo, you can use an LED + resistor on D8.
    • Double-check grounds: Arduino GND must be common with the sensor/relay module ground.
  2. Software Upload
    • Open the Arduino IDE, paste the sketch from the Arduino Code section, select Arduino UNO, and upload.
  3. Adjust Threshold
    • Adjust the threshold value in the sketch to match your ambient light conditions.
  4. Test
    • Cover the LDR to simulate darkness — the lamp should switch ON. Uncover to switch OFF.

Arduino Sketch

Below is a tested example sketch that reads an LDR on A0 and toggles a lamp/relay on D8. Replace the threshold or pins to match your wiring.

// Automatic Night Lamp System
// LDR on A0 (voltage divider). Relay or LED on D8.

const int ldrPin = A0;      // LDR connected to analog pin A0
const int lampPin = 8;      // Relay or LED connected to digital pin 8
int threshold = 500;        // Adjust this threshold for your environment (0-1023)
unsigned long debounceDelay = 200; // ms

unsigned long lastToggle = 0;
int lampState = LOW;

void setup() {
  Serial.begin(9600);
  pinMode(lampPin, OUTPUT);
  digitalWrite(lampPin, lampState);
}

void loop() {
  int sensorValue = analogRead(ldrPin);
  Serial.print("LDR: "); Serial.println(sensorValue);

  // Simple hysteresis: only toggle after debounceDelay
  if (millis() - lastToggle > debounceDelay) {
    if (sensorValue < threshold && lampState == LOW) {
      lampState = HIGH;            // it's dark -> turn lamp ON
      digitalWrite(lampPin, lampState);
      lastToggle = millis();
    } else if (sensorValue >= threshold && lampState == HIGH) {
      lampState = LOW;             // it's bright -> turn lamp OFF
      digitalWrite(lampPin, lampState);
      lastToggle = millis();
    }
  }

  delay(150);
}

Tip: If you're using a relay module, ensure the relay coil supply and the Arduino share a common ground. For mains lamps, always follow safety rules and disconnect power while wiring.
