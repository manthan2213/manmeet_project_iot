Smart Lighting Control

An IoT project that automatically controls lighting based on ambient conditions, using Arduino and light sensors.

Project Overview

Our Automatic Night Lamp System uses an LDR (Light Dependent Resistor) to detect ambient light levels and automatically control lighting. Perfect for energy saving and hands-free operation.

Components Used

  • Mini Breadboard For prototyping
  • Jumper Wires Male & Female
  • Resistor For voltage division
  • Arduino UNO Main controller
  • LDR Sensor Light detection
  • LED/Lamp Output device

Build Instructions

1. Hardware Setup

  • Connect LDR to Arduino analog pin A0
  • Use voltage divider with resistor
  • Connect LED/relay to digital pin D8
  • Verify power connections

2. Software Upload

  • Open Arduino IDE
  • Load the provided sketch
  • Select Arduino UNO board
  • Upload code

3. Testing

  • Power up the system
  • Cover LDR to test dark state
  • Expose to light to test bright state
  • Adjust threshold if needed

Arduino Code

// Automatic Night Lamp System
// LDR on A0 (voltage divider). Relay or LED on D8.

const int ldrPin = A0;      // LDR connected to analog pin A0
const int lampPin = 8;      // Relay or LED connected to digital pin 8
int threshold = 500;        // Adjust this threshold for your environment (0-1023)
unsigned long debounceDelay = 200; // ms

unsigned long lastToggle = 0;
int lampState = LOW;

void setup() {
  Serial.begin(9600);
  pinMode(lampPin, OUTPUT);
  digitalWrite(lampPin, lampState);
}

void loop() {
  int sensorValue = analogRead(ldrPin);
  Serial.print("LDR: "); Serial.println(sensorValue);

  // Simple hysteresis: only toggle after debounceDelay
  if (millis() - lastToggle > debounceDelay) {
    if (sensorValue < threshold && lampState == LOW) {
      lampState = HIGH;            // it's dark -> turn lamp ON
      digitalWrite(lampPin, lampState);
      lastToggle = millis();
    } else if (sensorValue >= threshold && lampState == HIGH) {
      lampState = LOW;             // it's bright -> turn lamp OFF
      digitalWrite(lampPin, lampState);
      lastToggle = millis();
    }
  }

  delay(150);
}
