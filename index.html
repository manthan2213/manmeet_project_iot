Introduction

This project is designed to automatically control a lamp based on the surrounding light intensity. It uses an LDR (Light Dependent Resistor) sensor to continuously monitor the brightness of the environment. When the light level drops below a certain threshold — such as during evening or in dark conditions — the Arduino UNO processes the sensor data and switches the lamp ON through a relay module (or an LED for demonstration purposes). Similarly, when sufficient daylight is detected, the lamp automatically turns OFF. This system provides an efficient, hands-free solution for smart lighting control. It helps conserve energy by ensuring that lights are only used when necessary, making it ideal for use in homes, streets, and offices. Moreover, it serves as an excellent beginner-level IoT and automation project, helping learners understand how sensors, microcontrollers, and electronic components interact to create real-world applications.

Step-by-step Build Instructions

  1. Hardware Wiring
    • Place the Arduino and the mini breadboard.
    • Connect one leg of the LDR to 5V and the other leg to analog pin A0 via a resistor to ground (voltage divider).
    • Wire the relay module (or the lamp driver) to a digital pin (e.g., D8) and to an external power supply if the lamp needs higher voltage/current. For demo, you can use an LED + resistor on D8.
    • Double-check grounds: Arduino GND must be common with the sensor/relay module ground.
  2. Software Upload
    • Open the Arduino IDE, paste the sketch from the Arduino Code section, select Arduino UNO, and upload.
  3. Adjust Threshold
    • Adjust the threshold value in the sketch to match your ambient light conditions.
  4. Test
    • Cover the LDR to simulate darkness — the lamp should switch ON. Uncover to switch OFF.

Arduino Sketch

Below is a tested example sketch that reads an LDR on A0 and toggles a lamp/relay on D8. Replace the threshold or pins to match your wiring.

// Automatic Night Lamp System
// C++ code
//
void setup()
{
  Serial.begin(9600);
  pinMode(4, OUTPUT);
}

void loop()
{
  int Val = analogRead(A2);
  Serial.println(Val);
  if (Val < 500) {
    digitalWrite(4, LOW);
  }
  else {
    digitalWrite(4, HIGH);
  }
}

Tip: If you're using a relay module, ensure the relay coil supply and the Arduino share a common ground. For mains lamps, always follow safety rules and disconnect power while wiring.
